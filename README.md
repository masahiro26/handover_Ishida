# 運動実行を伴う運動想起課題におけるリアルタイム脳波解析に関する研究

## 概要
R8年度修了の石田将大の研究。

## 内容
### 背景
ブレインコンピュータインターフェース（BCI）は、脳活動を利用して外部機器を制御する技術であり、四肢運動に制約を持つ患者の意思伝達やリハビリテーション支援への応用が期待されている。特に脳波（EEG）を用いた非侵襲型BCIでは、身体を動かさずに脳活動を誘発できる運動想起（Motor Imagery）が重要な研究対象となっている。
しかし、運動実行と運動想起を同時に行う条件下での研究は限られており、さらにリアルタイムでの脳波解析による判別については十分に検討されていないという課題がある

## 研究内容
本研究では、右手タッピング運動を実行しながら、右腕または左腕の伸縮運動を想起する課題を対象として、脳波データをリアルタイムで解析するシステムを構築した。
Biosemi ActiveTwoによる脳波計測データをPCへ逐次転送し、Pythonを用いて前処理、特徴量抽出、機械学習による判別（SVM）を連続的に実行することで、運動想起部位の違いをリアルタイムで識別することを試みた。
